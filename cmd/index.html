Invidious2NewPipe

Invidious2NewPipe is a tool for converting your Invidious OPML exports into a format that NewPipe understands, allowing you to share your subscriptions between the two services.

Links generated by this service do not last forever, but should last more than enough time for you to re-use the link to import into multiple devices

  1. Visit the settings page on your Invidious instance
  2. Export subscriptions as OPML (for NewPipe & FreeTube)
  3. Upload the file using this form
  4. Share the resulting link to the device with NewPipe installed
  5. Download and import the json file
